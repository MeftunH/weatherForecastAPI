## About this project
main goal is to write an API to communicate with these APIs. Let's show the daily, weekly and monthly weather report according to the country / city information that we will get from the user.

The request will first come to the API you will write, then after you receive the request, you will request the Free Weather APIs and return this information.

1. [✅] Making a correct API structure. 
2. [✅ ] Making validations on your models or parameters.
3. [✅ ] Handling exceptions.
